1) Eclipse Project (BookContentProvider_00001.rar) with an Android 
application that implements a ContentProvider. The provider is
wrapped around an SQLite database (book_info.db) with four tables:
book_title, book_author, book_cover_image, and book_author_image.
Details of the tables are in CPBookDbAdptr.java.

2) The ContentProvider is implemented in BookContentProvider_00001.java.

3) Targets: Android 2.3+, Eclipse Helios 3.6

4) To start: 1) unzip into a folder; 2) open in Eclipse; 3) Run
as Android application to install the content provider.
